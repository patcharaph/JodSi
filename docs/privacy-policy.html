📋 Introduction

JodSi (จดสิ) is an AI Voice-to-Note application that converts voice recordings into summarized notes. We prioritize your privacy above all else.

This policy is effective as of March 1, 2026.

📦 Data We Collect

Data Type Details When
Anonymous ID Auto-generated on app launch, not tied to identity First app open
Audio Files .m4a voice recordings you create When you record
Transcripts Speech-to-text results from your recordings After recording
AI Summaries Key Takeaways, Details, Action Items After transcription
Profile (if linked) Name, email, profile photo from LINE/Google When you link an account

🔒 How We Use Your Data

  • Transcription — Audio files are sent to Deepgram API for speech-to-text conversion
  • AI Summarization — Transcribed text is sent to Google Gemini API for summarization
  • Storage — Notes and audio are stored on Supabase (secure servers) so you can access them
  • Service Improvement — Aggregate, anonymized usage analytics to improve the app

⚠️ We do NOT sell your data to third parties.

🤝 Third-Party Services

Service Purpose Data Shared
Supabase Auth, Database, Storage User data, audio files, notes
Deepgram Speech-to-Text Audio file (URL)
Google Gemini AI Summarization Transcribed text

Each service has its own privacy policy. Please refer to their respective websites for details.

👤 Anonymous Accounts

When you open the app, an Anonymous ID is created automatically — no sign-up required. You can use all features without creating an account.

  • Anonymous accounts inactive for more than 90 days are automatically deleted along with all associated data
  • If you uninstall the app or switch devices without linking an account, your data cannot be recovered
  • You can link a LINE or Google account at any time to preserve your data

🛡️ Security

  • Data is encrypted in transit (TLS/SSL)
  • Database uses Row Level Security (RLS) — users can only see their own data
  • API keys are stored securely on the server, never embedded in the app

✅ Your Rights

  • Access — View your notes, audio, and summaries anytime through the app
  • Deletion — Delete individual notes or your entire account
  • Export — Copy summaries as text at any time
  • Opt-out — Stop using the app at any time. Anonymous data is auto-deleted within 90 days

👶 Children

This app is not designed for children under 13. We do not knowingly collect data from children. If a parent discovers their child is using this app, please contact us to have the data removed.

📝 Policy Changes

We may update this policy from time to time. If there are significant changes, we will notify you through the application.
